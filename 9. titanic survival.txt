Title of the Assignment:
Use the inbuilt dataset ‚Äòtitanic‚Äô. Plot a box plot for distribution of age with respect to each gender
along with the information about whether they survived or not.

Objective:
To visualize how passenger age varies with gender and survival status using a box plot.

------------------------------------------------------------
CODE:
------------------------------------------------------------
# Step 1: Import Libraries
import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

# Step 2: Load Titanic Dataset
titanic = sns.load_dataset('titanic')
print("First 5 rows of dataset:")
print(titanic.head())

# Step 3: Check for Missing Values
print("\nMissing values in 'age' column:", titanic['age'].isnull().sum())

# Step 4: Drop Missing Age Values (for simplicity)
titanic = titanic.dropna(subset=['age'])

# Step 5: Plot Box Plot
plt.figure(figsize=(8,6))
sns.boxplot(x='sex', y='age', hue='survived', data=titanic, palette='coolwarm')
plt.title('Age Distribution by Gender and Survival Status')
plt.xlabel('Gender')
plt.ylabel('Age')
plt.legend(title='Survived', loc='upper right', labels=['No (0)', 'Yes (1)'])
plt.show()

VIVA QUESTIONS:

Q1. What does a box plot represent?
‚Üí It shows the distribution of data based on five summary statistics: minimum, Q1, median, Q3, and maximum.

Q2. What do the whiskers in a box plot indicate?
‚Üí They show the spread of data beyond the quartiles; points outside are considered outliers.

Q3. Why do we use hue in Seaborn boxplots?
‚Üí To compare distributions between multiple categorical groups.

Q4. What is the inference from the Titanic box plot?
‚Üí Younger females had a higher survival rate compared to males.

Q5. What is an outlier?
‚Üí A data point that lies significantly outside the normal range of values.

Q6. Why are females shown to have higher survival?
‚Üí Due to priority evacuation policy ("women and children first") during the disaster.
